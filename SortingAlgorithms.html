Sorting Algorithms

A sorting algorithm is exactly what the name suggests, an algorithm that sorts a set of data (typically in to numerical or alphabetical order). All sorting algorithms are not made equal though! Some are better than others depending on the situation. Certain algorithms are better on partially sorted data, others may not be quite as fast but are much easier to implement. To learn more about the speeds of algorithms and how they are classified see my page on Big O Notation

Below are a list of common sorting algorithms:

  • Bubble Sort
  • Quick Sort
  • Merge Sort
  • Insertion Sort
